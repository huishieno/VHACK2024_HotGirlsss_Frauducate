Frauducate
---------------
An online educational platform to tackle the pervasive issue of financial fraud by offering 
comprehensive resources and educational materials. Our platform aims are to raise 
awareness and equip individuals with the knowledge and skills needed to prevent 
falling victim to fraudulent activities.

We provide various type of fraud prevention courses, simulations with real-world scenarios, quizzes,
and guides to report fraud activities.


Languages used:
---------------
- html
- css
- scss
- javascript
